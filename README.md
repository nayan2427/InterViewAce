# InterViewAce 🎯

A browser-based interview preparation tool that helps you practice technical questions under timed, realistic conditions — built with pure HTML, CSS, and JavaScript, no frameworks or backend needed.

![InterViewAce Screenshot](screenshot.png)
> *Replace `screenshot.png` with your actual screenshot file*

---

## Features

- 📚 Categorized technical Q&A sets (DSA, OOP, Web concepts)
- ⏱️ Timed self-assessment mode to simulate real interview pressure
- ✅ Instant score feedback after each session
- 📱 Fully responsive — works on mobile and desktop
- ⚡ Zero dependencies — runs entirely in the browser

## Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Page structure & semantic markup |
| CSS3 + Flexbox | Responsive layout & styling |
| JavaScript (ES6+) | DOM manipulation, event handling, score logic |

## Getting Started

No installation needed. Just clone and open:

```bash
git clone https://github.com/nayan2427/interviewace.git
cd interviewace
# Open index.html in your browser
```

Or simply download the ZIP and open `index.html`.

## How It Works

1. Select a question category
2. Answer each question within the time limit
3. Submit to see your score and correct answers
4. Repeat to improve

## Project Structure
